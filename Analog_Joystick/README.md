# Analog_Joystick
Using Blech to read commands of a joystick connected to the analogue pins A1+A2 and the digital D2 (for clicking).

As always.. do

```
$ cd Analog_Joystick
$ make
```

and then build the project using PlatformIO in vscode (the main file is src/main.cpp).
